A modern Android weather application built with Jetpack Compose, delivering real-time forecasts, interactive maps, and detailed environmental insights in a clean and responsive UI.

---

## Overview

Raindar provides accurate, location-based weather data combined with rich visualisation tools. It is designed to balance simplicity with depth, making it suitable for both quick checks and detailed analysis.

---

## Features

- Real-time weather data and air quality insights  
- 5-day forecast with 3-hour intervals  
- Location-based weather using device GPS  
- Global city search  
- Interactive weather maps with multiple data layers  
- Dark and light mode support  
- Smooth, responsive UI with modern design principles  

---

## Tech Stack

- Kotlin  
- Jetpack Compose  
- MVVM Architecture  
- Retrofit & OkHttp  
- Mapbox SDK  
- OpenWeather API  
- Room Database  

---

## Architecture

The app follows an MVVM architecture to ensure clear separation of concerns and scalability:

- View: Jetpack Compose UI components  
- ViewModel: State management and business logic  
- Model: API responses and local data  

---

## Screens

- Home – Current weather, air quality, and search  
- Map – Interactive weather layers and visualisations  
- Forecast – Detailed multi-day forecast view  
- Settings – Theme and preferences  

---

## Getting Started

1. Clone the repository  
2. Add your API keys (OpenWeather, Mapbox)  
3. Open in Android Studio  
4. Run on an emulator or device  

---

## Project Structure

- api/ – Network layer and data models  
- screens/ – UI screens  
- maps/ – Map-related components  
- nav/ – Navigation logic  
- utils/ – Shared helper functions  

---
