#### Development Framework
The app will be developed using Kotlin and Jetpack Compose for a modern,
flexible UI.

#### System Architecture
A MVVM (Model-View-ViewModel) architecture will be used for eﬃcient data
binding and separation of concerns.
APIs and Libraries:
- OpenWeather API: For real-time weather and air quality data.
- Mapbox: For interactive map rendering and custom weather layers.
- Retrofit Client: For eﬃcient API communication.
- OkHTTP: For handling weather layers
- Local Storage: To improve performance and allow oﬄine access, the app
will use Room Database to cache frequently accessed weather data.
